## :information_source: Como baixar o projeto

```bash
#Clonar o repositório
$ git clone https://github.com/CurtisYoung/JavaScriptEs6.git

# Entrar no diretório
$ cd JavaScriptEs6

# Instalar as dependências
$ yarn install

# Inicializar o projeto
$ yarn dev
```
